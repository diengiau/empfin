Chapter 2 Data Management

This chapter provide some idea about how to manage your precious data when doing research.

First, I will cover some types of data files, how to import and export, and how to efficiently provide data to your coauthors. Second, I explain some important issues when you manipulate data such as missing data, summarize data, and some common errors.
